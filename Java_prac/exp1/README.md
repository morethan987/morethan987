# 基于命令行的学生成绩管理系统

采用 MVC 模式，使用 Java 语言实现的一个简单的学生成绩管理系统。功能特色：

1. 基于命令行界面，易于使用和部署
2. 支持多角色权限控制，包括管理员、老师和学生
3. 仿照数据库设计思路，使用本地文件实现了数据的持久化存储

## 常用命令

```bash
# 测试代码
mvn test -Dtest=MainTest

# 运行程序
mvn compile && mvn exec:java
```
